데이터 웨어하우스 구축 : 기초

 비즈니스 인텔리전스(BI) 는 최근 들어 역동적이고 빠르게 발전했고 많은 사람들은 그로부터 이점을 얻는 중입니다. 이 새로운 BI의 성능을 가장 성공적이고 효율적인 방법으로 사용할수 있으려면 기본적으로 , 수중의 모든 데이터를 담고 분석할 수 있어야 합니다. 데이터 웨어하우스에 입문해 보세요.

 간단히 말하자면 데이터 웨어하우스는 비즈니스에 관련된 다양한 기원을 가지는 거대한 데이터의 저장소입니다. 데이터 웨어하우스는 트랜색션 데이터베이스(OLTP 시스템 ; Online transaction systems) 가 트랜색션에 집중할 수 있도록 데이터 분석 업무(OLAP;Online Analytic processing) 에 주로 쓰입니다. 유의미하게 많은 데이터가 한 장소에 저장되어 있다면 비즈니스를 분석하고 개선된 숙의를 행사 할 수 있을 것입니다.

 데이터 웨어하우스에 접근하는 법은 다양하지만, 이 튜토리얼은 구축에 필요한 기초적 지식에 집중할 것입니다.


데이더 웨어하우스 구축의 필요성

 논쟁의 여지가 있지만 모든 데이터를 한점에 집중시키는 것은 데이터 웨어하우스의 가장 큰 이점 중 하나입니다. 그 외의 장점들을 알아 보겠습니다.

    - 시간 절약 : 비즈니스 이용자들은 데이터 웨어하우스에 존재하는 다양한 출처의 정보에 빠르게 접근할 수 있습니다, 다른말로 하자면 다양한 출처의 정보를 찾느라 시간 낭비를 하지 않게 된다는 뜻입니다.

    - 신뢰도 향상 : 데이터 웨어하우스에 전송되는 데이터는 인력에 의존하는 방법과는 반대로 정제되고 최신의 완전한 데이터라는 뜻입니다. 이러한 데이터들로 인해 더욱 높은 신뢰도를 얻을 수 있습니다.

    - 통찰력 향상 : 데이터 웨어하우스는 소유중인 데이터들을 더 쉽게 분석할수 있도록 조직화합니다.

    - 보안성 향상 : 데이터에 접근할 때 중앙화된 연결점이 있다면 데이터 접근 권한을 쉽게 관리할 수 있습니다. 데이터 웨어하우스는 보안을 온전히 필요에 맞추어 조절할 수 있게 함으로서 데이터 소유자는 의도한 대로 접근 권한을 줄 수 있고 다른 시스템으로부터 고립시킬 수 있습니다.

 데이터 웨어하우스의 잠재력은 무궁무진합니다, 데이터 분석이 더욱 편리해지고 비즈니스에 중요한 질의에 답변하기는 더욱 쉬워집니다. 정리되고 유효한 데이터는 답변을 더욱 빠르고 안전하게 취할 수 있게 합니다.
